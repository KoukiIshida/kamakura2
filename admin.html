管理ページ

ここで設定した内容は、この端末のブラウザに下書きとして自動保存されます。
参加者に配るには「settings.json を書き出す」でファイルを保存し、それを GitHub のリポジトリに置いて（差し替えて）ください。
※ 答えは書き出し時に SHA-256 でハッシュ化され、原文はファイルに残りません（＝ソースを見ても答えはわかりません）。
※ 一度書き出した settings.json を読み込んで再編集する場合、答えは伏せられています。変更したい答えだけ入力し直してください（空欄なら元の答えを維持します）。
💡 問題文に「差し込みタグ」が使えます（参加者のニックネームに応じて自動で置き換わります）：
{ニックネーム} … ニックネーム全体／ {呼び名} … ニックネーム＋呼び方（例：たろうくん）／ {1文字目} … 1文字目（数字を変えて {2文字目} なども可）／ {最後の文字}／ {文字数}
例：問題文に 答えの1文字目は「{1文字目}」 と書くと、ニックネーム「たろう」の人には 答えの1文字目は「た」 と表示されます。
ニックネーム登録ルール

参加者が入力したニックネームがこのルールを満たさない場合、はじいて再入力を促します。

オンにすると、問題文やクリア画面で {呼び名}（例：たろうくん）が使えます。ニックネーム本体（文字数・文字種）とは別の設定です。
